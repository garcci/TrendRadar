# 文章生成 Prompt 补充要求

本文件存储文章生成的动态 Prompt 要求，由自动进化系统根据文章质量趋势自动调整。

## 基础内容要求

- 聚焦AI、开源、芯片、科技商业
- 不聊政治流水账、娱乐八卦
- 科技内容占比 ≥ 70%
- 全文不超过 1500 字
- 禁止重复输出同一章节
- 禁止复制输入的热点标题当观点
- 禁止编造虚假数据（表格数据必须来自输入热点）
- 禁止模板化废话
- 正文中不要再次出现"快速阅读区"

## 输出格式规范

### 快速阅读区
- 一句话总结：20-40字概括核心科技趋势
- 阅读时间估算
- 核心观点：5条洞察（不要复制热点标题）
- 关键词：3-6个具体中文技术词

### 深度分析
- 2-3个分析板块
- 每个板块：小标题概括核心观点 + 技术原理解释 + 趋势预测
- 每个板块包含1个表格 + 1个关键洞察引用块
- 引用输入热点中的真实数据，不要编造数字

### 平台热点精选
- 格式：`平台名：[标题](链接) - 一句话点评`
- 精选5-8个最有价值的平台热点

### 趋势观察
- 3-4条趋势性观察
